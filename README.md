# rexchao1.github.io

My personal site — [rexchao1.github.io](https://rexchao1.github.io).

Built with Next.js (static export) and deployed to GitHub Pages via the
workflow in `.github/workflows/deploy.yml`. Every push to `main` rebuilds and
publishes.

## Local development

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # static export into ./out
```
